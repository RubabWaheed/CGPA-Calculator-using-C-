# CGPA-Calculator-using-C-
A clean C++ console application for calculating SGPA and CGPA. Supports 1-8 semesters with grade validation and formatted output. Built with pure C++ standard library - no external dependencies.  Features: - SGPA calculator (single semester) - CGPA calculator (multi-semester, 1-8) - 13 grade variants (A+ to F) -
